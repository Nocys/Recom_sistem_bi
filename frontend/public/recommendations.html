Analisis Rekomendasi

Halaman ini menampilkan ringkasan teknis metode rekomendasi yang digunakan sistem.

Content-Based Filtering

TF-IDF + Cosine Similarity

Metadata produk diproses menjadi token, dihitung dengan TF-IDF, lalu dibandingkan memakai cosine similarity.

category material material_variant style_theme dominant_color room_category

Harga tidak digunakan dalam CBF karena sistem fokus pada kemiripan item, bukan transaksi.

User-Based Collaborative Filtering

Cosine Similarity

User dibandingkan berdasarkan Implicit Preference Score pada user-product matrix.

Like aktif

1.0

Wishlist aktif

1.5

Pernah tanya admin

2.5

implicit_score(u, i)
= (1.0 x active_like)
+ (1.5 x active_wishlist)
+ (2.5 x has_whatsapp_inquiry)

Page view tidak digunakan dalam CF. Like dan wishlist memakai state aktif, sedangkan tanya admin dihitung pernah/tidak.

Hybrid Recommendation

70% CBF + 30% CF 
Final Score = (0.7 x CBF Score) + (0.3 x CF Score)

CBF Score dan CF Score dinormalisasi pada rentang 0 sampai 1 sebelum digabungkan.

Room-Based Complementary Recommendation

Rule ruangan
  • Kamar Tidur -> Bedset, Lemari, Nakas, Meja Rias
  • Ruang Tamu -> Living Set, Meja, Kursi
  • Ruang Makan -> Meja, Kursi
  • Dapur -> Kitchen Set, Meja, Kursi
  • Mini Bar -> Minibar, Kursi, Meja
Complementary Score
= (0.45 x room_match)
+ (0.35 x complementary_category_match)
+ (0.15 x style_match)
+ (0.05 x color_match)

Harga tidak digunakan dalam complementary recommendation.
